# Projects

This folder contains the main active open-source projects developed by **Lady Legalz** and **TUOLAT LABZ**. Each project is designed to empower paralegals and legal professionals with practical, ethical, and accessible technology tools.

## Active Projects

| Project                  | Description                                                                 | Status     | Key Features                          | Folder/Link                          |
|--------------------------|-----------------------------------------------------------------------------|------------|---------------------------------------|--------------------------------------|
| **lady-legalz**          | Core Python library for paralegal-friendly legal tech tools                 | Active     | Case summarization, contract analysis, redaction, evidence hashing | Coming Soon  |
| **LegalAI Assistant**    | AI-powered chat assistant for legal research and explanations               | In Development | Local/offline Q&A, transparent responses | Coming soon                          |
| **Digital Courtroom**    | Web-based virtual courtroom prototype with video and document sharing       | Prototype  | WebRTC integration, mock hearings     | Coming Soon      |
| **ContractGen**          | Natural language to contract generator                                      | Active     | Template-based clause generation     | Coming Soon                          |
| **CaseLaw Summarizer**   | Standalone tool for extracting holdings and reasoning from court opinions   | Active     | Integrated in `lady-legalz` library   |Coming Soon|

## Project Standards

All projects follow these principles:
- **Open Source** – MIT licensed, free to use and modify
- **Ethical by Design** – Transparency, no hidden AI decisions, bias-aware
- **Paralegal-First** – Simple interfaces, no advanced coding required
- **Lightweight** – Minimal dependencies where possible
- **Accessible** – Supports diverse users (multilingual potential, screen-reader friendly)
