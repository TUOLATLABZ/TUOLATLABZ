## Key Guidelines for Using and Contributing

1. **Naming Conventions**  
   - Use clear, descriptive names: e.g., `Motion_for_Writ_of_Mandamus_Compensatory_Damages_Template.docx`.  
   - Include dates for versions: `YYYY-MM-DD_FileName_VersionX.ext`.  
   - Avoid spaces; use hyphens or underscores. No special characters.  
   - Prefix sensitive files with `[CONFIDENTIAL]` (e.g., `[CONFIDENTIAL]_EIV_Report_Analysis.pdf`).

2. **File Types and Best Practices**  
   - Preferred formats: `.md` (Markdown for guides), `.docx`/`.pdf` (legal docs), `.py` (tech scripts if related).  
   - Keep files under 10MB; compress large PDFs.  
   - Add metadata: Include a header in docs with author, date, purpose (e.g., "Drafted by Lady Legalz Member - For TUOLAT JD Tech Program").  
   - Version control: Use Git branches for major edits (e.g., `git checkout -b lady-legalz-update-motion`).

3. **Contribution Rules**  
   - **Fork & Pull Request Workflow**: Don't edit directly‚Äîfork the repo, make changes in your branch, then submit a PR with a clear description (e.g., "Added template for immigration fraud statutes reference").  
   - Review process: At least one other Lady Legalz member or Innovation Lab admin must approve.  
   - Cite sources: For legal refs (e.g., 18 U.S.C. ¬ß 1546 on fraud), link to official sources like Cornell LII.  
   - Inclusivity: All content must promote diversity, equity, and respect‚Äîfocus on empowering women in law/tech.

4. **Security & Confidentiality**  
   - Mark sensitive docs as private (use GitHub's file-level restrictions if needed).  
   - No real client data: Anonymize everything.  
   - Report issues: Use GitHub Issues for bugs or suggestions (label as "lady-legalz-docs").

5. **Tools & Integration**  
   - Link to TUOLAT resources: Embed links to [tuolatjuris.org](https://eshawnscip.wixsite.com/tuolatjuris) or Innovation Lab email.  
   - Collaborate: Share via Discord/Slack channels for Lady Legalz discussions.  
   - Update regularly: Review quarterly for 2026 relevance (e.g., add AI ethics guidelines).
